# Algo Templates by Romanov Sergey M3117

**Algo Templates** — это расширение для Visual Studio Code, которое позволяет быстро вставлять в редактор шаблоны кода для популярных алгоритмов.

Основная идея расширения — предоставить удобный инструмент для соревновательного программирования и изучения алгоритмов, избавляя от необходимости писать один и тот же код с нуля.

## Features

*   **Быстрая вставка кода**: Вставляйте готовые реализации алгоритмов через палитру команд.
*   **Полная настраиваемость**: Добавляйте, изменяйте или удаляйте шаблоны алгоритмов прямо в настройках VS Code (`settings.json`).
*   **Поддержка разных языков**: Изначально настроено для C++, но вы можете легко добавить шаблоны для Python, Java, JavaScript и других языков.

## Как использовать

1.  Откройте палитру команд (`Shift+Cmd+P` на macOS или `Shift+Ctrl+P` на Windows/Linux).
2.  Начните вводить и выберите команду **"Algo-Templates: Вставить шаблон алгоритма"**.
3.  В появившемся списке выберите нужный алгоритм.
4.  Код шаблона будет вставлен в текущий файл в позицию курсора.



## Настройки расширения

Это расширение добавляет следующие настройки в VS Code:

*   `algo-templates.cpp.snippets`: Объект, содержащий шаблоны для языка C++.

Вы можете изменять стандартные шаблоны или добавлять свои прямо в файле `settings.json`.

### Пример добавления нового шаблона

Чтобы добавить свой шаблон (например, для алгоритма поиска в глубину - DFS), откройте `settings.json` и добавьте новый объект в `algo-templates.cpp.snippets`:

```json
"algo-templates.cpp.snippets": {
    // ...существующие шаблоны
    "Depth First Search (DFS)": {
      "prefix": "algo_dfs",
      "body": [
        "void dfs(int u, const std::vector<std::vector<int>>& adj, std::vector<bool>& visited) {",
        "    visited[u] = true;",
        "    // std::cout << u << \" \";",
        "    for (int v : adj[u]) {",
        "        if (!visited[v]) {",
        "            dfs(v, adj, visited);",
        "        }",
        "    }",
        "}"
      ],
      "description": "Поиск в глубину (Depth First Search)"
    }
}
```

### Доступные алгоритмы (по умолчанию)

По умолчанию для C++ доступны следующие шаблоны:

*   Сортировка слиянием (Merge Sort)
*   Классический бинарный поиск (Binary Search)
*   Быстрая сортировка (Quick Sort)
*   Сортировка вставками (Insertion Sort)
*   Пирамидальная сортировка (Heap Sort)
*   Сортировка подсчетом (Counting Sort)
